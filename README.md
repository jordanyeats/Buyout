# Buyout

**A strategy game about building, investing in, and merging tech startups.**

Play it here: `https://[your-username].github.io/buyout/`

## How to Host

1. Create a new GitHub repository called `buyout`
2. Upload `index.html` to the repository
3. Go to **Settings → Pages**
4. Under "Source", select **Deploy from a branch**
5. Select the **main** branch and **/ (root)** folder
6. Click **Save**
7. Your game will be live at `https://[your-username].github.io/buyout/` within a few minutes

## How to Play

2–6 players compete to accumulate the most wealth by founding tech startups, buying equity, and orchestrating mergers. Place tiles to grow companies on a 9×9 board. When companies collide, mergers happen — and that's where the real decisions begin.

**Share blocks** represent 1,000 shares of a company. Prices rise exponentially as companies grow, capped at $100,000 per block.

**Merger cards** (optional) add unpredictability — a 27-card deck drawn when mergers trigger. Some help, some hurt, and two are devastating.

The game ends when any company reaches size 35. Highest net worth wins.

## Features

- 2–6 players (human + AI opponents)
- Three AI difficulty levels (Random, Greedy, Strategic)
- Optional 27-card merger deck with 15 unique effects
- $100K share price cap for balanced late-game strategy
- Works on desktop and mobile browsers
- No server required — runs entirely in the browser
